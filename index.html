WHAT IS HACKTOBERFEST?

Hacktoberfest is a month-long celebration of open source software. Maintainers are invited to guide would-be contributors toward issues that will help move the project forward, and contributors get the opportunity to give back to both projects they like and others they've just discovered. No contribution is too small — bug fixes and documentation updates are valid ways of participating.

OPEN TO EVERYONE

Hacktoberfest is open to everyone in our global community. Whether you’re a developer, or a student learner you can help drive growth of open source and make positive contributions to an ever-growing community.

NO HARDCORE REQUIREMENTS

All backgrounds and skill levels are encouraged to complete the challenge.

MONTH LONG CHALLENGE

You can sign in anytime between October 5 to October 31, 2021 to participate and complete the challenge.

GIT & GITHUB

Git is software for tracking changes in any set of files, usually used for coordinating work among programmers collaboratively developing source code during software development. Its goals include speed, data integrity, and support for distributed, non-linear workflows. GitHub, Inc. is a provider of Internet hosting for software development and version control using Git. It offers the distributed version control and source code management functionality of Git, plus its own features.
